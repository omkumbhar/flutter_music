# flutter_music

To develop new options for current music applications which are simple,
convenient,with rich UI and consume less memory where users can do tasks easily.
The application is based of Flutter, technology which is cross platform
development UI tool kit having a great number of widgets for mobile application,
uses Dart as a programming language.The new application mainly realizes six core
functions including main play interface, playlists, menus, add music, file browsing
and song search. This player has merits of high performance, simple operation, and
runs independently on any mobile devices. The player can also browse and access
files in mobile phones with appropriate permissions
